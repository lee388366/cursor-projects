# Web Projects

Web 开发项目集合，包含前端和后端开发文件。

## 项目内容

### 泉州旅游规划网站
- `quanzhou_travel_plan.html` - 泉州旅游规划页面
- 包含完整的旅游路线规划和美食推荐

### Node.js 后端项目
- `middleware.ts` - TypeScript 中间件
- `package.json` - 项目依赖配置
- `package-lock.json` - 依赖锁定文件
- `node_modules/` - 依赖包目录

## 技术栈

- **前端**: HTML, CSS, JavaScript
- **后端**: Node.js, TypeScript
- **依赖管理**: npm

## 项目结构

```
web-projects/
├── README.md
├── quanzhou_travel_plan.html  # 旅游规划页面
├── middleware.ts              # TypeScript 中间件
├── package.json              # 项目配置
├── package-lock.json         # 依赖锁定
└── node_modules/             # 依赖包
```
